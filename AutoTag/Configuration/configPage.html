Quick Start Guide

Get up and running in 4 steps
1
Get API Keys
Trakt: Go to My Apps. New App -> Name it "AutoTag".
⚠️ Redirect URI: Set to anything, example http://localhost.
MDBList: Go to Preferences. Scroll down to "API Access".
2
Add a Rule
Click the "+ Add New Source" button. A new configuration block will appear at the top.
3
Configure
• Tag Name: The tag that will appear in Emby (e.g., "Trending").
• URL: Copy the full URL of a public list from Trakt or MDBList.
• Limit: (Optional) Limit how many movies to tag.
4
Save & Sync
Click "Save Settings" first. Then click the blue "Run Full Sync" button to start the process immediately.
5
Schedule
Edit the scheduled run in Embys "Scheduled Tasks".
